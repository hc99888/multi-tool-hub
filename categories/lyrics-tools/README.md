# 📝 歌词处理工具

本分类包含各种歌词处理和分析工具。

## 📂 项目列表

### 1. [歌词智能分析](./lyrics-analyzer/) ⭐
- **技术栈**：Next.js + React + TypeScript
- **功能**：
  - 歌词结构拆解
  - 爆款歌词分析
  - 意象词生成
- **启动**：`pnpm install && pnpm run dev`
- **文档**：[BUILD_APK.md](./lyrics-analyzer/BUILD_APK.md) | [README](./lyrics-analyzer/README.md)
- **状态**：✅ 可用

### 2. [歌词匠](./lyrics-crafter/)
- 功能：专业的歌词创作工具
- 状态：开发中

### 3. [歌词解析](./lyrics-parser/)
- 功能：深度解析歌词结构和含义
- 状态：开发中

## 🚀 快速开始

### 使用歌词智能分析

```bash
cd categories/lyrics-tools/lyrics-analyzer
pnpm install
pnpm run dev
```

访问 http://localhost:5000

## 💡 功能特色

### 歌词智能分析
- 📊 结构化分析：拆解歌词的段落、韵脚、句式
- 🎯 爆款分析：分析热门歌曲的成功要素
- 🎨 意象生成：为歌词创作提供灵感
- 📱 支持APK：可打包成Android应用
