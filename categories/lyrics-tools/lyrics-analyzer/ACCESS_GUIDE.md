# 如何访问应用

## 🌐 访问地址

服务正在运行！您可以通过以下方式访问：

### 方式一：使用预览界面（推荐）

通常在您使用的Coze开发界面中，会有一个**预览按钮**或**预览链接**，点击即可打开应用。

### 方式二：通过内网地址

如果您在同一网络环境中：
```
http://9.129.89.180:5000
```

### 方式三：检查开发平台

大多数在线开发平台（如Coze）会提供：
- 预览窗口/预览标签
- 分享按钮
- 部署URL

## 🔍 如何找到正确的访问地址

### 如果您在Coze平台：

1. **查看页面顶部或底部**
   - 可能有"预览"、"预览网页"、"打开预览"等按钮

2. **查看侧边栏**
   - 可能有"部署"、"预览"选项

3. **查看输出面板**
   - 可能有显示访问URL的信息

4. **查找"分享"或"公开"按钮**
   - 可能有生成公开链接的选项

## ❓ 如果还是无法访问

### 可能的原因：

1. **需要配置端口映射**
   - 某些平台需要在设置中配置端口5000的映射

2. **需要部署到公网**
   - 当前可能只在沙箱内部可访问
   - 需要部署到生产环境

3. **网络限制**
   - 防火墙或安全组规则可能阻止访问

### 解决方案：

**方案A：通过平台部署**

1. 在Coze平台找到"部署"或"发布"按钮
2. 按照提示配置部署
3. 获取公网访问地址

**方案B：手动部署到其他平台**

如果您有自己的服务器，可以：

```bash
# 1. 将代码下载到本地
# 2. 在服务器上运行
pnpm install
pnpm run build
pnpm run start
```

**方案C：联系技术支持**

如果以上方法都不行，请联系平台技术支持，询问如何访问您的应用。

## 📱 手机访问

如果您想在手机上访问：

### 如果有公网地址：
直接在手机浏览器输入地址即可

### 如果是内网地址：
需要使用内网穿透工具（如frp、ngrok等）将服务暴露到公网
